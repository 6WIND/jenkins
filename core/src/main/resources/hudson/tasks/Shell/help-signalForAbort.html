Numeric value of signal to send to spawned process. Only valid on operating systems supporting UNIX-like signals. This parameter overrides "Signal sent on abort" parameter defined in jenkins system global configuration.

Supported signals are:

  • 1 (SIGHUP)
  • 2 (SIGINT)
  • 3 (SIGQUIT)
  • 4 (SIGILL)
  • 5 (SIGTRAP)
  • 6 (SIGABRT)
  • 10 (SIGUSR1)
  • 12 (SIGUSR2)
  • 15 (SIGTERM)

For signals not designed to terminate a process, note that a SIGTERM will be send after the number of seconds configured in "Max timeout per process abort" parameter
